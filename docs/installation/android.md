## Установка на Android

Для Android в качестве основного клиента мы рекомендуем использовать [AnyPortal](https://github.com/AnyPortal/AnyPortal). Это надежный и простой в настройке клиент.

### 1. Скачивание приложения

1. Скачайте и установите по [ссылке](https://github.com/AnyPortal/AnyPortal/releases/latest) APK клиента совместимый с вашим телефоном.

   ![android1]({{ site.baseurl }}/images/installation/android/1.png){: width="50%"}
2. Откройте приложение после установки.

### 2. Настройка подключения

На текущий момент клиент поддерживает только ручной способ настройки.

#### Ручная настройка

1. Скопируйте ссылку подписки для IOS из нашего [Telegram бота](https://t.me/bez_filtrov_vpn_bot).  
   *(Раздел «Мое подключение» -> «Соединение <ID>»)*.
2. В приложении AnyPortal на главном экране перейдите во вкладку «**Профили**» нажмите «**︙**» в правом верхнем углу.

   ![android2]({{ site.baseurl }}/images/installation/android/2.png){: width="50%"}
3. В появившемся меню выберите «**Добавить группу профилей**».

   ![android3]({{ site.baseurl }}/images/installation/android/3.png){: width="50%"}
4. Вставьте ссылку, которую скопировали в пункте 1 в качестве значения URL и нажмите «**Сохранить и обновить**»

   ![android4]({{ site.baseurl }}/images/installation/android/4.png){: width="50%"}

5. Выберите созданный профиль как активный.

   ![android5]({{ site.baseurl }}/images/installation/android/5.png){: width="50%"}
6. Вернитесь в «**Панеле управления**» и запустите VPN с помощью кнопки «**Пуск**»

   ![android6]({{ site.baseurl }}/images/installation/android/6.png){: width="50%"}
