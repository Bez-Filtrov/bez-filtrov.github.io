## Обновление конфигурации

После обновления сессии необходимо обновить конфигурацию в приложении.

### Android

Откройте приложение [V2rayTun]("v2raytun://open).

<div style="display: flex; gap: 10px; justify-content: center;">
  <img src="{{ site.baseurl }}/images/session_update/android/1.jpg" alt="android1" style="width: 48%;">
  <img src="{{ site.baseurl }}/images/session_update/android/2.jpg" alt="android2" style="width: 48%;">
</div>

### IOS

На IOS нет возможности обновить вручную конфигурацию, но можно выставить автоматическое обновление при открытии приложения.

Откройте приложение [Streisand]("streisand://open).

1. Перейдите в «**Настройки**».

    ![iphone1]({{ site.baseurl }}/images/session_update/iphone/1.png){: width="50%"}
2. Откройте раздел «**Подписка**».

    ![macos2]({{ site.baseurl }}/images/session_update/iphone/2.png){: width="50%"}
3. Включите пункт «**Обновлять при открытии**».

    ![macos3]({{ site.baseurl }}/images/session_update/iphone/3.png){: width="50%"}
4. Перезапустите приложение.
